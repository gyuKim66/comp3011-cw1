# Frontend Application

This directory contains the **Next.js frontend** for the Weather Analytics platform.

The frontend provides a dashboard interface for viewing weather data, managing locations, and exploring analytics.

---

## Technology

- Next.js
- TypeScript
- React

---

## Run the Frontend

Install dependencies:

```bash
npm install
```

Start the development server:

``` bash
npm run dev
```


The application will be available at: `http://localhost:3000`



## Features

* Weather dashboard

* Featured locations display

* Location search and registration

* Temperature and humidity analytics

* Temperature trend charts


## Notes

This frontend consumes REST APIs provided by the FastAPI backend.
